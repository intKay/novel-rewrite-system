| 文件 | 修改 |
|------|------|
| `ollama_client.py` | 新增 `num_ctx` 构造参数（默认 4096），写入 `options.num_ctx` |

| `cli.py` | 新增 `--ollama-num-ctx` 参数（默认 2048），real 模式打印 `[诊断]` 信息 |

## 10. 失败原因

prompt 超长（风格报告全文 + 初稿片段 + 角色表 + 改写指令）导致 Ollama qwen runner 在 4GB 显存下 KV cache 溢出，返回 502 而非显式 OOM 错误。
