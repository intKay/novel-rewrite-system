## 输入

### 参考文本（约 350 字）

```
夜色沉下来。她推开门，听见远处的钟声。院中无人，只有几片枯叶在青石板上翻卷。

她缓步走向庭中老槐，手指抚过粗糙树皮，心中涌起千般思绪。这座宅院她住了十年，每一块砖、每一片瓦她都熟悉，可今夜月光下的院落却显得陌生起来。

忽然，身后传来一声轻咳。她没有回头，只淡淡说道："既然来了，何必躲藏。"

树影中走出一个青衣男子，约莫三十岁上下，腰间挂着一柄长剑。他抱拳施礼："在下奉师命，特来拜会。今夜之事，还请姑娘三思。"

她终于转过身，目光清冷如霜："你家师叔的账，我还没跟他算。你倒是先来了。"夜风骤起，满院落叶盘旋而上，在两人之间形成一道无形的墙。青衣男子后退半步，手已搭上剑柄。
```

### 用户需求

```
创作一个悬疑推理短篇：退休刑警在旧案中找到新线索，发现真凶就在身边。风格冷硬写实，对话简洁，节奏紧凑。
```

## 命令

```bash
python -m novel_rewrite_system.cli --model-mode cloud \
    -r reference.txt \
    -q "创作一个悬疑推理短篇：退休刑警在旧案中找到新线索，发现真凶就在身边。风格冷硬写实，对话简洁，节奏紧凑。" \
    -p ./acceptance-6.3 \
    --chunk-chars 2000
```

**改写稿**：在原片段基础上增加了环境细节、动作描写和心理暗示，文笔更细腻。附带 6 条修改说明，逐条标注改动内容和理由。
