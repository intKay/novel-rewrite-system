## 新增接口

### `CreativeTag`（frozen dataclass）

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| `name` | `str` | 标签名 |
| `category` | `str` | 分类：情绪 / 节奏 / 人物关系 / 文风 / 剧情功能 |
| `description` | `str` | 标签解释 |
| `prompt_hint` | `str` | 传给模型的创作控制提示 |
| `safety_note` | `str` | 边界说明（可选） |
| `fallback_hint` | `str` | 模型拒绝时的替代表达（可选） |

### `get_default_tags() -> list[CreativeTag]`

返回 12 个默认标签，覆盖全部 5 个分类。

### `format_tags_for_prompt(tags: list[CreativeTag]) -> str`

- 空列表返回空字符串
- 选中标签格式化为以 `## 创作控制标签` 开头的 Markdown 文本
- 按 `CATEGORIES` 顺序分组
- fallback_hint 以 "（若具体表达受限，请改用：...）" 形式附在 prompt_hint 后

### 续写接口扩展

- `build_continuation_prompt` 新增 `tag_hints` 参数
- `generate_continuation` 新增 `selected_tags` 参数
- `run_webui_continuation` 新增 `selected_tags` 参数

## 数据流

```
WebUI 多选标签
  → run_webui_continuation(selected_tags=[...])
    → generate_continuation(selected_tags=[...])
      → format_tags_for_prompt(selected_tags) → tag_hints string
        → build_continuation_prompt(tag_hints=tag_hints)
          → 模板中插入 "## 创作控制标签" 区域
            → ModelClient.generate(request)
```
