## 配置变化

- 新增运行依赖：`streamlit>=1.30`。
- `cloud-only` 模式需要环境变量 `DEEPSEEK_API_KEY`。
- `fake` 模式无需 API key、无需网络、无需本地 GPU。

## 手动验收步骤

1. 启动：`streamlit run src/novel_rewrite_system/webui.py`。
2. 默认选择 `fake`，填写项目路径、参考文本、用户创作需求，点击运行。
3. 确认页面显示运行完成、风格分析、初稿、改写结果和输出文件路径。
4. 未设置 `DEEPSEEK_API_KEY` 时选择 `cloud-only`，确认页面显示清楚的 API key 缺失错误和建议。
5. 设置 `DEEPSEEK_API_KEY` 后，可用短文本小样例人工验收 `cloud-only`。
6. 验收过程中不需要 Ollama，不访问本地 GPU。
