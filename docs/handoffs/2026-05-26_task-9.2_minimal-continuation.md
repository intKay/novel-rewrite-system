## 新增接口

### `build_continuation_prompt(current_text, continuation_instruction, *, style_reference=None, target_word_count=None) -> ModelRequest`

- 存于 `generation/prompts/continuation.py`
- 纯函数，不调用模型
- 系统 prompt 要求承接当前文本、保持一致、不重复、不附带解释
- 模板包含：当前文本、扩写要求、可选风格参考、可选字数目标

### `generate_continuation(current_text, continuation_instruction, model_client, *, style_reference=None, target_word_count=500, project_path=None) -> ContinuationResult`

- 存于 `generation/continuation_service.py`
- 验证输入 → 构建 prompt → 调用模型 → 返回结果
- 可选保存到 `drafts/continuation.md`

### `run_webui_continuation(current_text, continuation_instruction, mode, *, style_reference=None, deepseek_model, project_path, target_word_count=500) -> WebUiContinuationResult`

- 存于 `webui.py`
- 复用 `_build_model_clients` 处理 fake / cloud-only 模式切换
- 输入验证 → 调用 `generate_continuation` → 返回结果

### 数据结构

- `ContinuationResult(continuation_text: str)` — frozen dataclass
- `WebUiContinuationResult(ok, continuation_text, error_code, error_message, error_suggestion)` — frozen dataclass

## WebUI 变化

- 生成成功后结果保存到 `st.session_state.pipeline_result`
- 结果展示区和续写区从 session_state 读取，跨 Streamlit 重绘维持
- 续写区展示：扩写需求 text_area + "生成下一段" 按钮 + 续写结果展示
- 页面文案更新为："填写创作需求即可开始生成。生成后可在结果区继续扩写当前片段。"
