## 暴露接口

- `PipelineResult` — frozen dataclass，12 个字段
- `run_minimal_pipeline(reference_text, user_requirement_text, project_path, *, source_id, model_client, local_client, rewrite_instruction, chunk_chars) -> PipelineResult`
- 导入路径：`from novel_rewrite_system.pipeline import PipelineResult, run_minimal_pipeline`

## 数据流

```
reference_text
    → clean_text()
    → split_text() → chunks
    → analyze_style(chunks, model_client, project_path=project) → StyleAnalysisResult
    → StoryRequirement.from_text(user_requirement_text) → StoryRequirement
    → generate_draft(style_report, requirement, model_client, project_path=project) → DraftGenerationResult
    → _extract_first_fragment(draft_text, chunk_chars) → fragment
    → rewrite_locally(..., fragment, rewrite_instruction, ..., local_client, project_path=project) → LocalRewriteResult
    → _write_pipeline_log(project, summary, error_log)
    → PipelineResult
```

## PipelineResult 字段

| 字段 | 类型 | 说明 |
|------|------|------|
| `style_report` | `str` | 风格分析报告 |
| `style_source_count` | `int` | 参考文本来源数 |
| `style_chunk_count` | `int` | 切片数量 |
| `requirement_raw` | `str` | 用户原始需求文本 |
| `draft_text` | `str` | 初稿正文 |
| `rewritten_fragment` | `str` | 改写后片段 |
| `rewrite_original_fragment` | `str` | 被改写的原初稿片段 |
| `project_path` | `Path` | 项目根目录 |
| `style_report_path` | `Path` | analysis/style_analysis_report.md |
| `draft_path` | `Path` | drafts/draft.md |
| `rewritten_path` | `Path` | revisions/rewritten.md |
| `error_log` | `list[str]` | 非致命错误摘要 |

## 输出文件

| 文件 | 路径 |
|------|------|
| 风格分析报告 | `{project_path}/analysis/style_analysis_report.md` |
| 初稿 | `{project_path}/drafts/draft.md` |
| 改写稿 | `{project_path}/revisions/rewritten.md` |
| 管道日志 | `{project_path}/logs/pipeline.log` |

## 首个片段提取规则

1. 若初稿含"第X章"标记 → 取第一章内容
2. 否则取前 `chunk_chars` 字符，在句号/问号/感叹号边界截断
3. 若无句子边界 → 在 `chunk_chars` 处硬截并补句号
