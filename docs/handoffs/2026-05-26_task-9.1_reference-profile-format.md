## 新增文档

### `docs/reference_profile_format.md`

定义了用户为小说资料来源整理"参考画像文档"时应遵循的 Markdown 格式规范。包含：

1. **目的** — 说明画像文档不是原文全文，而是供模型理解的压缩参考
2. **适用范围** — 当前仅用于 WebUI 单文件上传 / 粘贴
3. **推荐结构（13 节）**：
   - 基本信息、故事类型与基调、语言风格、叙事节奏、人物设定、人物关系、情节结构、典型场景、常用表达与意象、禁止偏离点、可借鉴点、不应直接复制的内容、给生成模型的使用说明
4. **每节规范** — 分别说明应该写什么、不应该写什么、推荐长度
5. **完整原创示例** — 虚构武侠悬疑《江湖夜雨录》参考画像，约 180 行
6. **使用建议** — 长度控制、写作策略、隐私注意事项
7. **后续扩展说明** — 设定摘要、人物卡、标签词库、续写上下文、自动画像生成

## 验收方式

人工验收：用户阅读 `docs/reference_profile_format.md` 后，应能独立将手头的小说资料整理为一个结构完整的参考画像 Markdown 文件，并通过 WebUI 文件上传功能传入系统。
