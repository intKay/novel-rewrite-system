## 新增/修改的代码

### `_DEFAULT_REWRITE_INSTRUCTION`（模块常量）

```python
_DEFAULT_REWRITE_INSTRUCTION = (
    "请在不改变核心剧情和人物设定的前提下，润色当前片段，使描写更自然、节奏更流畅。"
)
```

用于替代旧默认值 `"请用更细腻的描写改写当前片段"`。`run_webui_pipeline` 的 `rewrite_instruction` 参数默认值改为该常量。

### UI 变化

- 主表单新增 `st.caption`："填写创作需求即可开始生成。生成后会自动产出一个润色改写片段。未来版本将支持更灵活的二次改写与扩写。"
- 高级参数中改写指令入口标签改为"内部默认改写指令（可选）"，默认值使用 `_DEFAULT_REWRITE_INSTRUCTION`。
- `run_webui_pipeline` 公开接口不变。
