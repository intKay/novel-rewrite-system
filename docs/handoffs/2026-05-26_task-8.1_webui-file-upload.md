## 新增/修改函数

### `_resolve_reference_text(uploaded_bytes, manual_text) -> str`

- 内部辅助函数，不属于公开接口。
- `uploaded_bytes: bytes | None` — 上传文件的字节内容，`None` 表示未上传。
- `manual_text: str` — 用户手动粘贴的文本。
- 返回最终参考文本：有上传文件则使用文件内容，否则使用手动文本。
- 文件 UTF-8 解码失败时抛出 `ValueError`（用户可读中文消息）。
- 空上传文件返回空字符串（后续 pipeline 的 `EmptyTextError` 稳定触发）。

## 数据流变化

```
WebUI 表单
  ├─ st.text_area("参考文本") ──→ manual_text ──┐
  ├─ st.file_uploader("上传 .txt") ──→ uploaded_bytes ──→ _resolve_reference_text ──→ ref_text ──→ run_webui_pipeline(reference_text=ref_text)
  └─ 点击"运行" ──→ 先解析后调用
```

优先级：上传文件内容 > 手动粘贴文本。
